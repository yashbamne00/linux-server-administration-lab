# Linux Server Administration Lab

## Project Overview

The **Linux Server Administration Lab** is a hands-on project designed to simulate the daily responsibilities of a Linux System Administrator. The lab was built using **Ubuntu Server 22.04 LTS** running inside **Oracle VirtualBox**, providing a safe environment to practice real-world server administration tasks.

The primary goal of this project is to gain practical experience with Linux by configuring, securing, and managing a server from the command line. Throughout the project, I performed common administrative tasks such as managing users and groups, configuring file permissions, securing remote access with SSH, installing and managing web servers, configuring a firewall, scheduling automated tasks, monitoring system logs, and writing basic Bash scripts for automation.

Rather than focusing only on Linux commands, this project emphasizes understanding how different system components work together to build and maintain a functional server environment.
